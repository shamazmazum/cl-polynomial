Overview

cl-polynomial is a library for factoring polynomials in one variable over finite fields with a prime number of elements \(p\). NB: Many provided functions require \(p\) as their last argument. No special tests are done to check if \(p\) is really a prime and behaviour is undefined if it is not (e.g. some functions can simply become non-terminating).

Few examples: 

CL-USER> (polynomial:sequence->polynomial #*1110010101001)
X^12 + X^9 + X^7 + X^5 + X^2 + X^1 + X^0
CL-USER> (polynomial:factor * 11)
((1 . X^6 + 6X^5 + 4X^4 + 3X^3 + 8X^2 + 5X^1 + 9X^0)(1 . X^1 + 5X^0)
 (1 . X^5 + 10X^3 + X^0))
1

CL-USER> (polynomial:list->polynomial '(1 0 1 1 0 0 1 0 1 1))
X^9 + X^8 + X^6 + X^3 + X^2 + X^0
CL-USER> (polynomial:factor * 2)
((1 . X^3 + X^2 + X^0)(2 . X^2 + X^1 + X^0)(2 . X^1 + X^0))
1
