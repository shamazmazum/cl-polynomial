API

A name of a package with described functions is given in parentheses.

Utility functions (cl-polynomial/util)

mod-sym(n q)
Compute \(n \mod q\). The result is in a range \(0 \dots 1\) if \(q = 2\) or \(-(q-1)/2 \dots (q-1)/2\) if \(q > 2\).
invert-integer(n q)
Find a multiplicative inverse of \(n\) in \(\mathbb{Z}_q\), i.e. find \(x\) such that \(xn = nx = 1\). Signal an error, if there is no such inverse.
gcdex(u v)
For \(u, v \in \mathbb{Z}\) find \(\gcd(u,v)\) and solutions of an equation \(au + bv = \gcd(u,v)\) with minimal absolute values.

Basic operations on polynomials (cl-polynomial/polynomial)

list->polynomial(coeffs)
Given a list of integer coefficients convert it to a polynomial. An index into the list corresponds with a power for that coefficient, e.g.:
CL-USER> (polynomial:list->polynomial '(1 2))
#<POLYNOMIAL:POLYNOMIAL 2X + 1>
monomials->polynomial(list)
Convert a list of monomials in the form (degree . coefficient) to a polynomial. A list must not contain two monomials of the same degree.
sequence->polynomial(seq)
Like list->polynomial, but for all types of sequences.
CL-USER> (polynomial:sequence->polynomial #*101)
#<POLYNOMIAL:POLYNOMIAL X^2 + 1>
polynomial->list(polynomial)
Convert a polynomial to a list of coefficients. This is an inverse of list->polynomial.
polynomial=(poly1 poly2)
Test if two polynomials are equal.
polynomial/=(poly1 poly2)
Test if two polynomials are not equal.
degree(polynomial)
Return degree of a polynomial
leading-coeff(polynomial)
Return the leading coefficient of a polynomial
map-poly(fn polynomial)

Given a polynomial \(\sum_n a_n x^n\) return a polynomial \(\sum_n fn(a_n)x^n\).

This is fmap for a type data Poly a b = Poly [(a, b)].

derivative(polynomial)
Return a formal derivative of a polynomial.
monicp(polynomial)
Test if a polynomial is monic
constantp(polynomial)
Test if a polynomial has degree 0.
+one+
Multiplicative identity
+zero+
Additive identity
+variable+
The polynomial \(x\)

Algebraic operations for polynomials over a ring (cl-polynomial/polynomial)

negate(polynomial)

Given a polynomial \(\sum_n a_n x^n\) return a polynomial \(\sum_n -a_n x^n\).

This is function is equivalent to 

(alexandria:curry #'map-poly #'-)
add(&rest polynomials)
Add polynomials together
subtract(polynomial &rest polynomials)
Subtract polynomials from polynomial.
multiply(&rest polynomials)
Multiply polynomials
scale(polynomial c)
Multiply a polynomial by a constant.
expt(f n)
For polynomial \(f(x)\) and a non-negative integer number \(n\) calculate \(f^n(x)\).

Division and factorization in \(\mathbb{F}_p[x]\)(cl-polynomial/fpx)

modulo(polynomial q)
Return a polynomial with every coefficient of polynomial being taken modulo q.
divide(poly1 poly2 p)
Calculate \(p_1 / p_2\) where \(p_1, p_2 \in \mathbb{F}_p[x]\), \(p\) being prime. A quotient and a remainder are returned as 2 values.
remainder(poly1 poly2 p)

Calculate a remainder of \(p_1 / p_2\) where \(p_1, p_2 \in \mathbb{F}_p[x]\), \(p\) being prime.

This function returns the second value of divide.

gcd(poly1 poly2 p)
Calculate the greatest common divisor of two polynomials in \(\mathbb{F}_p[x]\), p being prime.
gcdex(poly1 poly2 p)
Find \(\gcd(p_1, p_2)\), \(p_1, p_2 \in \mathbb{F}_p[x]\) and also find a solution of Bezout's equation \(a p_1 + b p_2 = \gcd(p_1, p_2)\) with minimal possible degree of \(a\) and \(b\).

square-free(polynomial p)
Perform square-free factorization of a polynomial in \(\mathbb{F}_p[x]\), \(p\) being prime. A list of tuples \((d_i . f_i)\) is returned, so the supplied polynomial is equal to \(\prod_i f_i^{d_i}\) multiplied by the second returned value.
reducing-polynomials(f p)
Return a list of all f-reducing polynomials for a monic non-constant square-free polynomial \(f \in \mathbb{F}_p[x]\).
factor(polynomial p &optional (algorithm :berlekamp))
Factor a polynomial in \(\mathbb{F}_p[x]\) into irreducible factors. An optional parameter algorithm can be either :berlekamp(default) or :cantor-zassenhaus.
irreduciblep(f p)
Test if a polynomial \(f \in \mathbb{F}_p[x]\) is irreducible.

Factorization in \(\mathbb{Z}[x]\)(cl-polynomial/zx)

suitable-prime(polynomial)
Return a suitable prime for reducing a factorization in \(\mathbb{Z}[x]\) to a factorization in \(\mathbb{F}_p[x]\).
remove-content(f)
For polynomial \(f \in \mathbb{Z}[x]\) return its primitive part and content.
lift-factors(f gs p n)
For a monic polynomial \(f \in \mathbb{Z}[x]\) and a list of its factors in \(\mathbb{F}_p[x]\), lift these factors to \(\mathbb{Z}_{p^{2^n}}[x]\).
gcd(poly1 poly2)
Calculate GCD of two polynomials in \(\mathbb{Z}[x]\).
square-free(polynomial)
Perform square-free factorization for a polynomial \(f \in \mathbb{Z}[x]\). The first returned value is a list of tuples \((d_i . f_i)\), so the polynomial is equal to \(\sum_i f_i^{d_i}\) multiplied by a second returned value.
factor(polynomial)
Factor a polynomial in \(\mathbb{Z}[x]\).
irreduciblep(polynomial)
Test if polynomial is irreducible in \(\mathbb{Z}[x]\).
cyclotomic(n)
Get n-th cyclotomic polynomial in \(\mathbb{Z}[x]\).
